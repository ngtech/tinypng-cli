tinypng-cli.py
==============

A simple command line client for TinyPNG.

### Usage

    ./tinypng-cli.py <image.png>

output:

    input size: 203 kb
    output size: 62 kb
    output ratio: 0.3076
    output url: https://api.tinypng.com/output/4jjna274387m8o2k.png
